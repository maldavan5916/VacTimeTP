Тема: Выполнение и сопровождение организационных (производственных) задач, разработанных для организации  

Разработка форм программы  
В рамках выполнения организационных задач была проведена разработка форм пользовательского интерфейса программы. Основное внимание уделялось созданию удобного и интуитивно понятного интерфейса для работы с базой данных и функционалом программы.  

Паттерн проектирования:  
Для разработки программы был использован паттерн MVVM (Model-View-ViewModel).  

Краткое описание MVVM:  
- Model (Модель): Хранит бизнес-логику и данные приложения.  
- View (Представление): Отвечает за визуальное представление данных (формы, элементы интерфейса).  
- ViewModel (Модель представления): Связывает модель и представление. Реализует обработку команд и управление данными.  

Основные элементы разработки  
1. Технологии:  
   - WPF (Windows Presentation Foundation).  
   - Язык программирования C#.  
   - Использование привязки данных (Data Binding) для связи интерфейса с логикой.  
2. Разработанные компоненты программы:  
   - Главное окно:  
     - Использован `Window` как основной контейнер.  
     - Добавлены вкладки для навигации (`TabControl`).  
   - Таблицы данных:  
     - Для отображения данных использован `DataGrid`.  
     - Реализована сортировка и фильтрация данных.  
   - Формы ввода:  
     - Использованы `TextBox`, `ComboBox`, `DatePicker` для ввода данных.  
     - Кнопки (`Button`) для выполнения операций (добавление, редактирование, удаление).  
   - Диалоги подтверждения:  
     - Реализованы модальные окна с помощью `MessageBox` для подтверждения действий пользователя.  
3. Особенности реализации:  
   - Использован `INotifyPropertyChanged` для обновления интерфейса при изменении данных.  
   - Реализована команда через `RelayCommand` для выполнения действий (например, добавления записей).  
   - Все элементы интерфейса адаптированы для обработки исключений и удобства пользователя.
