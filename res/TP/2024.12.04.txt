Тема: Выполнение и сопровождение организационных (производственных) задач, разработанных для организации

Основной функционал
1. Страница `EmployeeDivisionReport.xaml`:
   - Реализована визуальная часть отчёта с использованием MaterialDesignInXaml для стилизации.
   - Добавлены элементы управления:
     - Кнопки для обновления, печати и изменения ориентации документа.
     - Элементы выбора типа группировки (по подразделению, по должности, без группировки).
     - Фильтры по должности и подразделению.
2. Код ViewModel (`EmployeeDivisionReportViewModel`):
   - Добавлены свойства и методы:
     - Фильтры: Выбор должности и подразделения через `ComboBox`.
     - Группировка: Реализовано три типа группировки.
     - Генерация отчёта: Создание таблиц с заголовками и данными в `FlowDocument`.
   - Поддержка динамического обновления данных на основе фильтров и группировки.
3. Код `EmployeeDivisionReport.xaml.cs`:
   - Связь страницы и ViewModel.
   - Поддержка кэширования данных и обработки переходов.

Дополнения к другим файлам
1. Обновление маршрутизации в `MainWindow.xaml.cs`:
   - Включён переход на новую страницу `EmployeeDivisionReport`.
2. Улучшения в `MaterialUsageReport.xaml.cs`:
   - Упрощён код и исправлены избыточные вызовы `SetProperty` в свойстве `SelectedProduct`.
   - Убрана фиктивная запись "Все" из списка продуктов.

Особенности реализации
1. Гибкость отчёта:
   - Пользователь может настроить отображение данных через выбор группировки или фильтров.
   - Отчёт автоматически обновляется при изменении фильтров.
2. Печать отчёта:
   - Реализована горячая клавиша Ctrl+P для печати.
   - Визуально настроен документ для удобства чтения.
3. Качество UI:
   - Применены стили из библиотеки Material Design.
   - Добавлены подсказки к кнопкам для улучшения пользовательского опыта.
