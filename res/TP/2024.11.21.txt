Тема: Выполнение и сопровождение организационных (производственных) задач, разработанных для организации  

Разработка программного обеспечения  

Основные изменения и доработки:  
1. Обновление темы приложения:  
   - В файле `App.xaml` были внесены изменения в параметры темы:  
     - Цветовая схема была изменена для более современного дизайна:  
       - Основной цвет (PrimaryColor) установлен как `#FFE64A1C`.  
       - Вторичный цвет (SecondaryColor) — `#03DAC6`.  
   - Использован компонент `MaterialDesign` для стилизации приложения.  
2. Кэширование страниц для навигации:  
   - В файле `MainWindow.xaml.cs` реализовано кэширование страниц для оптимизации работы приложения.  
     - Кэширование позволяет сохранять ранее открытые страницы в памяти, предотвращая их повторное создание.  
     - Новый словарь `_pagesCache` хранит соответствие между именем страницы (ключ) и объектом страницы (значение).  
   - Добавлены новые страницы, такие как `Unit`, `Contract`, `Division` и другие, для отображения данных.  
3. Изменения интерфейса страницы `Unit`:  
   - В файле `Unit.xaml`:   
     - Добавлены элементы для взаимодействия:  
       - Поле поиска (`TextBox`) с функцией фильтрации данных.  
       - Кнопки управления (`Добавить`, `Удалить`, `Сохранить`, `Отмена`).  
       - Лейбл для вывода сообщений о состоянии (успех, ошибка).  
     - Изменен стиль таблицы `DataGrid` с использованием MaterialDesign, улучшено визуальное отображение.  
   - В `Unit.xaml.cs`:  
     - Реализованы основные функции:  
       - Добавление новой строки с использованием контекста базы данных.  
       - Удаление выбранной строки.  
       - Сохранение изменений в базе данных.  
       - Отмена внесенных изменений с откатом состояния базы данных.  
       - Поиск и фильтрация записей по тексту из поля поиска.  
4. Функциональность:  
   - Добавление новой записи:  
     - При нажатии на кнопку «Добавить» создается новый объект типа `Unit`, который добавляется в базу данных.  
   - Удаление записи:  
     - Удаление выбранной строки из таблицы с последующим обновлением данных.  
   - Сохранение изменений:  
     - Все внесенные изменения сохраняются в базу данных через `SaveChanges`.  
   - Отмена изменений:  
     - Откат изменений в случае ошибки или отказа от сохранения.  
   - Поиск:  
     - Фильтрация записей в таблице на основе текста, введенного пользователем.  

Итог: Выполненные изменения улучшили производительность и удобство использования приложения. Оптимизация интерфейса и добавление кэширования страниц обеспечили плавность работы и современный дизайн. Функции управления данными позволяют пользователям эффективно взаимодействовать с базой данных.
