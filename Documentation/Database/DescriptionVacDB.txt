Справочники
- сотрудники - Employees (
			ФИО - fio, 
			*подразделение - Divisions_id, 
			*должность - Posts_id, 
			дата приёма - dateHire, 
			дата уволнени - dateDismissal
			
			Дата рождения (dateOfBirth) – для расчёта возраста и льгот.
			Телефон (phoneNumber)
			Адрес проживания (address)
			Паспортные данные (passportData) – для кадрового учёта.
			Оклад (salary) – базовая ставка сотрудника.
			Дополнительные выплаты (bonuses) – надбавки, премии.
			Банковские реквизиты (bankDetails) – для выплат зарплаты.
			)

- должности - Posts (
			name - наименование)

- изделия - Products (
			название - name, 
			заводской_номер - serialNo, 
			*единица_измерения - Units_id
			*место_хранения - Locations_id
			количество - count,
			цена - Price)

- единицы_измерения - Units (
			название - name
			описание - description)

- места_хранения - Locations (
			название - name, 
			*кладовщик - Employees_id)

- подразделения - Divisions (
			название - name)

- материалы\коплектуюшие - Materials (
			название - name, 
			*единица_измерения - Units_id, 
			*место_хранения - Locations_id, 
			количество - count, 
			цена - price)

Таблица связи "Изделие_Материалы" (Product_Materials):
			id (Primary Key)
			product_id (Foreign Key из таблицы "Изделия")
			material_id (Foreign Key из таблицы "Материалы")
			quantity (Количество материала для конкретного изделия, если нужно)

- контрагенты - Counterparties (
			name - ФИО/наименование
			legalAddress - юридический адрес
			phoneNomber - номер телефона
			postalAddress - почтовый адресс
			ynp - УНП
			bankAccount - расчётный счёт 
			type [fiz/yr] - тип [физ/юр]
			okulp - ОКЮЛП
			okpo - ОКПО
			oked - ОКЕД)

Документы
- договора - Contracts (
			Название - name
			*контрагент - Counterparties_id, 
			дата заключения - date, 
			срок(месацев) - Term,
			сумма - summ, 
			*изделие - Products_id, 
			количество - count)

- поступления - Receipts (
			*материал\комплектуюшие - Materials_id
			сумма - summ, 
			дата - date
			количество - count
			*контрагент - Counterparties_id)

- реализация - Sales (
			*договор - Contracts_id
			сумма - summ, 
			дата реализация - date
			количество - count)

Отчёты
1. Отчет по использованию материалов на изделия (MaterialUsageReport)
   Описание: Показывает, какие материалы использовались для производства определенных изделий, их количество и стоимость.
   Поля:  
     - Наименование изделия  
     - Наименование материала  
     - Количество материала  
     - Стоимость материала   

2. Отчет по сотрудникам и их подразделениям (EmployeeDivisionReport)
   Описание: Список всех сотрудников с указанием их подразделений, должностей, даты приема и даты увольнения (если есть).
   Поля:  
     - ФИО сотрудника  
     - Подразделение  
     - Должность  
     - Дата приема  
     - Дата увольнения  

3. Отчет по остаткам материалов и комплектуюших (StockBalanceReport)
   Описание: Информация о текущих остатках материалов и комплектуюших с указанием мест хранения.
   Поля:  
     - Наименование материала/комплектуюших  
     - Место хранения  
     - Количество  
     - Единица измерения  

4. Отчет по договорам с контрагентами (ContractorContractsReport)
   Описание: Информация о заключенных договорах с контрагентами, их суммах и изделиях.
   Поля:  
     - Название договора  
     - Контрагент  
     - Дата заключения  
     - Изделие  
     - Количество  
     - Сумма договора  

5. Отчет по реализации продукции (ProductSalesReport)
   Описание: Детализация продаж продукции: что было продано, кому, за какую сумму и в каком количестве.
   Поля:  
     - Дата реализации  
     - Контрагент  
     - Наименование изделия  
     - Количество  
     - Сумма реализации
